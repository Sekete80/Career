# Career Guidance and Employment Integration Platform - Starter

This is a starter skeleton for the project:
- Frontend: React (skeleton)
- Backend: Firebase Functions (example)
- Firestore security rules and sample config
- README, LICENSE and basic scripts

## What is included
- frontend/: React app skeleton with Login/Register + simple routing
- functions/: Firebase Functions example (admissions process + auth helper)
- firestore.rules: starter Firestore security rules
- firebase.json: basic Firebase config
- .gitignore

## Next steps
1. `cd frontend` -> `npm install` -> `npm start`
2. Install Firebase CLI and login: `npm install -g firebase-tools`, `firebase login`
3. Initialize Firebase in project and deploy functions / hosting as needed.
